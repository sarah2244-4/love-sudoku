Love Soduko

How to play:

  • The 9x9 grid contains 9 smaller 3x3 grids and you can only use the numbers 1-9
  • Each row must contain the numbers 1-9 once
  • Each column must contain the numbers 1-9 once
  • Each 3x3 grid must contain the numbers 1-9 once
  • Make sure you don't repeat any numbers!
  • Never try to guess
  • Try to beat your last time and wrong move score
